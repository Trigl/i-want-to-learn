Mac OSX 终端走 shadowsocks 代理

以zsh作为说明：

```
vim ~/.zshrc  
```

添加如下代理配置：

```
# proxy list
alias proxy_on='export all_proxy=socks5://127.0.0.1:1087'
alias proxy_off='unset all_proxy'
```

保存退出，使 .zshrc 文件改动生效：

```
source ~/.zshrc
```
